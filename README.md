# 🧠 FitForm AI

**FitForm AI** is a real-time AI-powered fitness web app that analyzes your exercise form using your camera and provides instant feedback to improve posture and prevent injury.

---

## 🚀 Features

* 🎥 Real-time pose detection using AI
* 💪 Push-up & squat form analysis
* 📊 Accurate rep counting (only valid reps)
* 🧠 Smart feedback system (depth, posture, speed)
* 📱 Mobile + Desktop responsive UI
* ⚡ High-performance (smooth real-time tracking)
* 🎯 Automatic exercise detection (pushup & squat)

---

## 🧠 Why FitForm AI?

Many people perform exercises incorrectly, leading to injuries or poor results.

👉 FitForm AI solves this by:

* Providing **real-time posture correction**
* Ensuring **only correct reps are counted**
* Acting like a **virtual fitness coach**

---

## 🧠 How It Works

1. Open the app
2. Allow camera access
3. Perform an exercise (push-up or squat)
4. Get real-time feedback and rep count
